### 3. 기술 스택
---
- **프레임워크**: Next.js

- **UI 라이브러리**: React

- **언어**: TypeScript

- **스타일링**: Tailwind CSS

- **데이터베이스**: Firebase
  
- **상태 관리**: Redux, TanStack Query
  
- **폼 관리**: React Hook Form, Zod
